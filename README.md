# NexusSearch - Inverted Index Engine in C

## 🚀 Overview

NexusSearch is a high-performance document search engine built using core data structures and information retrieval techniques.

## 🧠 Key Concepts

* Inverted Indexing
* Hash Tables + AVL Trees
* TF-IDF Ranking
* Positional Indexing (Phrase Search)

## ✨ Features

* 🔍 Fast word search
* 📄 Multi-document indexing
* 🧹 Stop-word removal
* 📊 Relevance scoring (TF-IDF)
* 🧩 Phrase search using positional matching

## 📂 Project Structure

```
main.c
docs/
  ├── doc1.txt
  ├── doc2.txt
```

## ⚙️ Compile & Run

```bash
gcc main.c -o nexussearch -lm
./nexussearch
```
